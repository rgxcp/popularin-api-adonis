## How to Use
1. Clone this repository to your desired location.
2. Create database and configure it on `.env` file.
3. Run migration `adonis migration:run`.
4. Run server `adonis serve --dev`.
5. Fire-up Postman.
6. Do-what-you-want-with-it!
